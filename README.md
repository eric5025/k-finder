# 🧡 k-finder / TravelLens

AI 기반 한국 기념품 인식 앱 프로젝트

## 📱 TravelLens 앱

외국인 관광객을 위한 AI 기반 모바일 앱으로, 사진을 찍으면 한국 기념품을 자동으로 인식하고 다국어 정보를 제공합니다.

**택스 리펀 스타일 주황색 브랜딩** 🧡 - 노이즈 마케팅을 위한 UI/UX 디자인

### ✨ 주요 기능
- 📸 AI 이미지 인식 (Perplexity AI)
- 🌍 5개 언어 지원 (한/영/일/중/스페인)
- 🗺️ 지도 연동 (카카오맵, 구글맵)
- 💾 검색 히스토리 저장
- ⭐ 즐겨찾기 기능

### 🛠️ 기술 스택
- React Native (Expo SDK 54)
- TypeScript
- Firebase
- Perplexity AI

```bash
cd TravelLens

# .env 파일에 API 키 입력
```
